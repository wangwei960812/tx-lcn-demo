# tx-lcn-demo
tx-lcn 分布式事务演示包括服务端tm和客户端tx
工程分为两个主要模块：分布式事务的客户端和服务端

服务端包括：
tm模块

客户端包括：
common-db 公共模块
spring-demo-client  微服务 分布式事务发起模块
spring-demo-d       微服务d 分布式事务参与方
spring-demo-e       微服务e 分布式事务参与方
